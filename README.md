# Pet Care Backend

Spring Boot REST API for a Pet Care Management System.

## Tech Stack

- Java 21
- Spring Boot
- Spring Security
- JWT Authentication
- Gradle
- REST API
- Docker
- Docker Compose
- PostgreSQL
- AWS EC2 (Deployment)
- Swagger / OpenAPI

## Features

- User management
- Pet management
- Care request management
- JWT-based login
- Protected REST APIs
- Validation and exception handling
- Dockerized backend application
